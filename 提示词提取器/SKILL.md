# 提示词提取器

从代码中提取并优化提示词，将自然语言需求转换为结构化的提示词模板。

## 核心能力

1. **提示词提取**：从代码注释、字符串或现有提示词中识别和提取提示词
2. **提示词优化**：改进提示词的结构、清晰度和效果
3. **提示词生成**：根据用户需求生成高质量的提示词模板
4. **格式转换**：支持多种提示词格式的转换

## 使用场景

- 从现有代码中提取分散的提示词
- 优化现有提示词的结构和表达
- 为新功能设计提示词
- 构建大语言模型应用时整理提示词模板

## 使用方法

```markdown
## 原始需求
[描述用户想要的提示词或需求]

## 提取的提示词
[结构化的提示词内容]

## 使用说明
[如何在该场景中使用该提示词]
```

## 最佳实践

1. 保持提示词简洁明了
2. 使用清晰的指令词（角色、任务、格式）
3. 包含必要的上下文信息
4. 指定输出的格式和结构
